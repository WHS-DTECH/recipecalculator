# Pre-Backend Integration: Navigation & Link Validation

- [ ] Click every navigation link in the navbar and sidebar (if present)
- [ ] Ensure all links go to the correct page or section
- [ ] No broken or dead links (404s)
- [ ] All external links open in a new tab (target="_blank") and have rel="noopener"
- [ ] Breadcrumbs (if present) are correct and functional
- [ ] All buttons and action links trigger the correct UI or modal
- [ ] No duplicate or confusing navigation options
- [ ] All navigation is accessible via keyboard (Tab, Enter)

---

Check off each item as you validate navigation and links. If you need help automating link checks or want a script to assist, just ask!
