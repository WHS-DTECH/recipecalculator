# Static Asset Checklist

- [ ] All images are optimized (compressed, correct format, no unused files)
- [ ] All icons are SVG or optimized PNG
- [ ] All font files are referenced via CDN or are optimized locally
- [ ] All asset paths are correct in HTML/CSS/JS
- [ ] Favicon is present and referenced in all HTML files
- [ ] No broken image or font links in the UI
- [ ] Remove any unused or duplicate assets from the project

---

Check off each item as you verify your static assets. If you need help with image optimization or asset management, just ask!
