# Accessibility & Responsiveness Review Checklist

- [ ] Run Lighthouse or axe accessibility audit on all main pages
- [ ] Fix any color contrast issues flagged by the audit
- [ ] Ensure all interactive elements (buttons, links, inputs) are keyboard accessible
- [ ] All forms have proper labels and ARIA attributes
- [ ] All images/icons have alt text or aria-hidden as appropriate
- [ ] Test navigation and forms with only the keyboard (Tab, Shift+Tab, Enter, Space)
- [ ] Test on mobile, tablet, and desktop (Chrome, Firefox, Edge, Safari)
- [ ] Check layout and touch targets on small screens
- [ ] No horizontal scrolling on mobile
- [ ] All modals, dropdowns, and menus are accessible via keyboard and screen reader

---

Check off each item as you verify accessibility and responsiveness. If you need help with any audit or fix, just ask!
