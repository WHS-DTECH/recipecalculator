Route Descriptions
  • recipes: View and manage all recipes (full recipe management page)
  • inventory: Food Room Inventory - Browse all recipes in grid or table view
  • shopping: Book the Shopping - Shopping list generator interface
  • booking: Class booking system for scheduling
  • admin: Administrative functions (staff/class upload, recipe management, permissions)
Important Notes
  • Admin role should always have access to all pages
  • Public Access role is assigned to all users by default
  • Specific users can be granted additional roles (Teacher, Technician, Admin)
  • Changes take effect immediately
  • Users will see "Access Denied" if they try to access a page without permission
